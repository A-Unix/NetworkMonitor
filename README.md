## Installation instructions(For linux)
 ### Clone the repo
  ```bash
  git clone https://github.com/A-Unix/NetworkMonitor.git
  ```
 ### Move to cloned folder
 ```bash
 cd NetworkMonitor
 ```
 ### Get permissions
  ```bash
  chmod +x setup.sh main.py
  ```
 ### Install dependencies
  ```bash
  ./setup.sh
  ```
 ### Run the main file
  ```bash
  python3 main.py
  ```
